Class BlockApi

Manages raw Block.

An IPFS Block is a byte sequence that represents an IPFS Object (i.e. serialized byte buffers). It is useful to talk about them as "blocks" in Bitswap and other things that do not care about what is being stored.

It is also possible to store arbitrary stuff using ipfs block put/get as the API does not check for proper IPFS Object formatting.

This may be very good or bad, we haven't decided yet Ø3DÞ04

This API is accessed via the Block property.
