Remarks

This API is accessed via the Pin property.

Pinned objects are locally stored and never garbage collected.

AddAsync(String, Boolean, CancellationToken)

Adds an IPFS object to the pinset and also stores it to the IPFS repo. pinset is the set of hashes currently pinned (not gc'able).

Declaration
public Task<PinnedObject[]> AddAsync(string hash, bool recursive = true, CancellationToken cancel = default (CancellationToken))
Parameters
Type Name Description
String hash

The String representation of a base58 encoded MultiHash.
Boolean recursive

true to recursively pin links of object; otherwise, false to only pin the specified object. Default is true.
CancellationToken cancel

Is used to stop the task. When cancelled, the TaskCanceledException is raised.
